E-Ticaret Satış Analizi Projesi

Bu proje, Python ve SQL kullanarak ham e-ticaret verilerini işleyen, veritabanına aktaran ve anlamlı içgörüler sunan uçtan uca bir veri analizi çalışmasıdır.

- Öne Çıkan Özellikler
- Veri İşleme: Pandas ile 3500+ satırlık verinin temizlenmesi.
- SQL Entegrasyonu: SQLite ile ilişkisel veritabanı mimarisi kurulumu.
- Trend Analizi: Satışların 2 yıllık zaman serisi analizi ve bölgelere göre satış verisi analizi.
- Görselleştirme: Seaborn ve Matplotlib ile profesyonel raporlama.
